**E-Commerce Sales Dashboard — Power BI**

Designed a two-page interactive sales analytics dashboard in Power BI, structured around an Executive Summary and a Detailed Dashboard view. Built a relational data model connecting Customers, Orders, Order Items, Products, and a custom Date dimension, with DAX measures (Total Sales, Total Orders, Average Order Value, Customer Count) powering KPI cards and visuals throughout.

The Executive Summary page gives a high-level view of business performance: monthly sales trend, product-level sales breakdown, category distribution, and a geographic sales map by state and city, with slicers for month, state, and category.

The Detailed Dashboard page enables deeper analysis with a multi-level pivot table (category → product → customer → month → sales), a clustered bar chart comparing top products by units sold, and a decomposition tree for root-cause exploration of order value by channel, payment method, shipping mode, and order status.

**Skills demonstrated:** Data modeling & relationships, DAX measures, multi-page report design, advanced Power BI visuals (decomposition tree, matrix/pivot table), interactive filtering, dashboard UX for both executive and analyst audiences.
